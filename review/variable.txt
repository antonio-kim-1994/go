- 변수(Variable) : 값을 저장하는 메모리 공간을 가리키는 이름

# 변수가 왜 중요한가 ?
- 프로그램이란 결국 데이터를 연산/조작 하는 일
- 변수를 통해 메모리에 접근하여 연산/조작을 수행한다

# 변수 선언(Variable Definition)
var a int = 10      
    => a : 우변의 값을 저장된 메모리 공간
    => int : type, = : 대입 연산자(assign)

# 변수 값 출력
fmt.Println(msg, a)
    => fmt : 함수
    => Println : 메서드
    => msg, a : 파라미터

# 변수의 유형
구조체(struct)
    => { }
