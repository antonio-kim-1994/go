%v : 데이터 타입에 맞춰서 기본 형태로 출력
%T : 데이터 타입을 출력
%t : boolean을 true/false로 출력
%d : 10진수 정숫값으로 출력(정수 타입만 가능)
%b : 2진수로 출력
%c : 유니코드 문자를 출력(정수 타입만 가능)
%o : 8진수로 출력
%O : 앞에 8진수임을 표시하는 0o를 붙여서 8진수로 값을 출력
%x : 16진수로 값을 출력. 10 이상 값을 a-f 소문자로 표시
%X : 16진수로 값을 출력. 10 이상 값을 A-F 대문자로 표시
%e %E : 지수 형태로 실숫값을 출력(실수 타입만 가능) => -1.234456e+78
%f %F : 지수 형태가 아닌 실숫값 그대로 출력(실수 타입만 가능, 소수점 6자리까지 표현) => 123.456
%g %G : 값이 큰 실숫값은 지수 형태(%e)로 출력, 작은 실숫값은 실숫값 그대로(%f) 출력
%s : 문자열 출력
%q : 특수 문자 기능을 동작하지 않고 문자열 그대로 출력
